启动方法
由于 package.json 是 "type": "module"，后端文件保存为 server.cjs（CommonJS 模式）。启动步骤：


# 终端 1：启动后端
node server/server.cjs       # → http://localhost:3000

# 终端 2：启动前端
npm run dev                   # → http://localhost:5173
