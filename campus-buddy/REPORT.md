# 项目名称：校园万能搭子 (Secure Edition)

## 安全措施总结

### 1. 数据脱敏

手机号属于个人隐私，不能完整显示在页面上。我在展示联系方式时用正则表达式 `(\d{3})\d{4}(\d{4})` 把中间四位替换成了 `****`，防止围观者直接看到完整号码。

### 2. XSS 防护

我的整个项目中**没有使用 `v-html`**，所有用户输入的内容都通过 Vue 的 `{{ }}` 插值语法渲染。Vue 会默认对内容进行转义，这样就算有人在输入框里写 `<script>alert('攻击')</script>`，也不会被执行。

### 3. SQL 注入防护

在新增搭子的 POST 接口中，我使用了**参数化查询**（`?` 占位符），把用户输入和 SQL 语句分开传递。这样恶意用户就算在标题里写 SQL 代码，也只会被当成普通文字存进去，不会被数据库执行。

### 4. 水平越权防护

删除按钮不是所有人都能看到的。我在详情页加了判断 `currentUser.id === currentBuddy.creatorId`，只有**发帖人本人**才能看到删除按钮。这样别人就不能越权删除不属于自己的帖子。

### 5. 防爆破设计

登录接口做了一个**3 秒冷却时间**。一旦输错密码，按钮就会被锁住 3 秒，防止有人用脚本暴力破解密码。3 秒虽然不长，但足以让自动化攻击变得非常低效。
